# 补录标准

## 基本字段

每条新增内容尽量补齐以下字段：

- 标题
- 类别
- 平台或来源
- 原始链接
- 发布时间
- 是否官方
- 一句话摘要
- 为什么值得收录
- 核验状态

## 核验状态建议

- `primary`
  直接来自官方账号、出版社、学校、主流媒体原始页
- `secondary`
  来自公开百科、转载报道、二次整理页，但仍可作为线索
- `to_verify`
  当前只有模糊线索，等待更权威来源补全

## 优先级顺序

1. 官方账号和原始发布页
2. 学校、出版社、公司官网
3. 权威媒体原始报道
4. 可交叉验证的百科和索引页

## 推荐写法

每条说明最好回答三个问题：

- 这是什么内容
- 它为什么重要
- 现在这个链接为什么值得保留

## 不建议的写法

- 只有标题，没有任何说明
- 只有转发链接，没有原始出处
- 只写“经典”“很火”，没有具体理由
- 把争议性结论写成已证实事实

## 图片与截图

- 优先保存封面图、视频卡面、书封和页面截图
- 如果截图来自新闻页或平台页，尽量在说明里注明来源
- 图片文件名建议与内容标题保持对应

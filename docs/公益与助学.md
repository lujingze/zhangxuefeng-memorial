# 公益与助学

除了考研、志愿填报和升学规划内容，公开资料中还能看到张雪峰老师在助学与公益方面留下的比较稳定的记录。这部分内容很适合放入纪念站，因为它能补足其公众形象中较少被系统整理的一面。

## 公开可核验的助学记录

### [关于开展郑州大学2024-2025学年“峰学蔚来助学金”评选工作的通知](https://www5.zzu.edu.cn/student/info/1003/11500.htm)

- 内容摘要：郑州大学公开通知“峰学蔚来助学金”评选工作安排。
- 为什么重要：这是学校官方页面，能直接证明张雪峰老师对母校助学项目的支持不是传闻，而是有正式制度化记录。
- 来源属性：郑州大学学生工作网官方页面。

### [郑州大学举办“峰学蔚来助学金”暨冬日送暖爱心物品发放仪式](https://edf.v.zzu.edu.cn/xwzx/15611.jhtml)

- 内容摘要：公开记录了助学金和物资发放活动。
- 为什么重要：相比制度通知，这类活动报道能更直观看到项目已实际执行。
- 来源属性：郑州大学教育发展基金会官方页面。

### [我校举办2025年度“峰学蔚来助学金”发放仪式暨座谈会](https://xgb.hlju.edu.cn/info/1034/1537.htm)

- 内容摘要：黑龙江大学公开报道同名助学金项目发放情况。
- 为什么重要：说明相关助学活动不只停留在单一学校，公开影响范围更广。
- 来源属性：黑龙江大学学生工作部官方页面。

## 为什么这部分值得单列

- 它补充了“张雪峰老师只讲现实和选择”的单一印象
- 它与很多视频里提到的普通家庭、托举与机会问题形成呼应
- 学校官方页面通常具备较高的可核验性，适合长期保存为纪念资料

# 经典语录

这页只整理有公开出处、能够指向原始页面的代表性表达。为了控制误引风险，优先采用公开视频标题、图书标题和公开报道中已经稳定出现的短句，而不收录来源不明的“网传语录”。

## 1. “选择比努力更重要”

- 出处类型：图书标题
- 对应条目：[《选择比努力更重要：名师张雪峰手把手教你填报高考志愿》](https://books.google.com/books/about/%E5%90%8D%E5%B8%88%E5%BC%A0%E9%9B%AA%E5%B3%B0%E6%89%8B%E6%8A%8A%E6%89%8B%E6%95%99%E4%BD%A0%E5%A1%AB%E6%8A%A5%E9%AB%98.html?id=_zdwzwEACAAJ)
- 为什么重要：这是张雪峰老师最具辨识度的核心表达，几乎可以视为其升学规划方法论的总标题。
- 延伸理解：他强调的并不是否定努力，而是提醒学生和家庭，路径判断本身就是竞争力的一部分。

## 2. “方向比努力更重要”

- 出处类型：图书标题
- 对应条目：[《方向比努力更重要：名师张雪峰考研通关攻略》](https://read.douban.com/ebook/315621097/)
- 为什么重要：这句话与“选择比努力更重要”形成呼应，更突出他在考研阶段的路径判断意识。
- 延伸理解：很多时候不是不够努力，而是投入方向和目标判断先出了问题。

## 3. “互联网！我回来了！”

- 出处类型：公开视频标题
- 对应条目：[《互联网！我回来了！》](https://www.bilibili.com/list/28152637?bvid=BV1b44y1w7Eo&oid=999330640)
- 为什么重要：这是一个具有明显个人风格的回归表达，也代表张雪峰老师在公众舆论场中的持续存在感。
- 延伸理解：它不仅是一句回归宣告，也带有与争议、关注和个人影响力正面相遇的意味。

## 4. “不服从调剂？后果自负”

- 出处类型：公开视频标题
- 对应条目：[《不服从调剂？后果自负》](https://www.bilibili.com/list/28152637?bvid=BV1m94y1H79S&oid=366195843)
- 为什么重要：这类短句最能体现张雪峰老师直给、强判断、强风险提示的表达风格。
- 延伸理解：它的核心不是制造焦虑，而是用更直接的话逼用户正视志愿填报中的现实风险。

## 5. “新闻和传播可不是一码事儿”

- 出处类型：公开视频标题中的核心表达
- 对应条目：[《千万注意，文科生不要瞎报！张雪峰：新闻和传播可不是一码事儿...》](https://www.bilibili.com/video/BV17i4y1f7i8/)
- 为什么重要：这是最具公共讨论度的一类表达，能直接体现他为什么会成为争议中心。
- 延伸理解：其影响并不只在专业选择本身，而在于它把“就业导向”和“教育理想”之间的冲突公开化了。

## 6. “你离考研成功，就差这本书”

- 出处类型：图书标题
- 对应条目：[《你离考研成功，就差这本书》](https://books.google.com/books/about/%E4%BD%A0%E7%A6%BB%E8%80%83%E7%A0%94%E6%88%90%E5%8A%9F_%E5%B0%B1%E5%B7%AE%E8%BF%99%E6%9C%AC%E4%B9%A6.html?id=IzRQswEACAAJ)
- 为什么重要：这是其考研时期最有传播力的标题表达之一，也体现出早期内容的强动员色彩。
- 延伸理解：相比后来的升学规划内容，这一阶段更偏“目标驱动”和“方法动员”。

## 7. “相信自己！你可以！”

- 出处类型：公开视频标题中的核心表达
- 对应条目：[《张雪峰：相信自己！你可以！建议所有高考生都来看看...》](https://www.bilibili.com/video/BV1NL4y1F7fz/)
- 为什么重要：这类表达让人看到张雪峰老师并不只有冷静的现实判断，也会在关键节点给出鼓励和情绪支持。
- 延伸理解：它补足了公众对其“只会讲现实”的单面印象。

## 这些表达为什么值得单列

- 它们高度浓缩了张雪峰老师的个人语言风格
- 它们能从不同阶段对应到考研、高考志愿、方法论和公众争议
- 它们比零散“名言摘抄”更容易核验、更适合长期公开保存

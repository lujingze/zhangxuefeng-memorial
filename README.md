## 仓库现在有什么

- 一个可直接用于 GitHub Pages 的静态站结构
- 一份人物传记与公开索引
- 经典视频、图书著作、时间线、公益与助学等专题页
- 账号、图书、视频、时间线、来源五类 Markdown 索引
- 面向后续补录的协作说明与补录标准

## 内容边界

- 收录公开可核验的账号、视频、图书、报道、学校页面与时间线资料
- 保留原始链接、摘要和纪念说明
- 不收录受版权保护内容的整段搬运

仓库已经补好 `_config.yml`、布局模板和站点样式。你只需要在仓库设置里开启 GitHub Pages，并把 Source 指向当前分支的根目录即可。
